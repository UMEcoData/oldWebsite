Michigan Chemical Heatmap


The goal of this project is to visualize where chemicals are above critical levels in Michigan. It is important to bring awareness to this information for Michigan residents because some chemicals that exist above critical levels can be dangerous. By compiling and structuring EPA UCMR datasets, we created a program that creates heatmaps displaying chemical levels searchable by chemical. For each chemical, we have written short summaries breaking down what it is and how it could potentially be harmful when above critical levels. The next step to this project is launching the program in an interactive web tool that users can easily filter by chemical and year.
